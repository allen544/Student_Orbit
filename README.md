## 📌 Overview
**Student Orbit** is a full‑stack student management system aimed at organizing campus activities efficiently.  
It helps students view academic schedules, RSVP to events, apply for absences, and manage their profile information.  
With a clean UI and responsive design, it ensures smooth access across devices.

---

## ✨ Key Features
- **📅 Dashboard** – Overview of all upcoming academic and personal activities.
- **⏰ Timetable** – Daily/weekly class schedule with easy navigation.
- **🎟 Event RSVP** – Register and track upcoming campus events.
- **📝 Absence Application** – Submit leave requests digitally.
- **📂 Profile Management** – Update academic and personal information.
